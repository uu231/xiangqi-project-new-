# 🎮 中国象棋游戏项目

## 📋 项目简介
基于Java Swing开发的中国象棋对战游戏，实现了完整的象棋规则和游戏逻辑。

## 🚀 运行项目
```bash
mvn exec:java -Dexec.mainClass="edu.sustech.xiangqi.XiangqiApplication"
```
